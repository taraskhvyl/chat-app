# Chat App

A real-time chat application built with Next.js, React, and TypeScript.

## Installation

```bash
yarn install
```

## Development

```bash
yarn dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser.

## Build

```bash
yarn build
yarn start
```

## Testing

```bash
# Unit tests
yarn test

# E2E tests
yarn test:e2e
```
